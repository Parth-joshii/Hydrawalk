## 📱 Mobile Installation Guide (Android)
1. Download **`app-release.apk`** from the link above.
2. Send the file to your phone.
3. Open it on your phone and tap **Install**.
4. If prompted with a warning (like *Play Protect / Blocked*), tap **"More Details"** and click **"Install Anyway"** (this is standard for custom applications built from source code).

---

## 🖥️ PC Installation Guide (Windows)
1. Download **`hydrawalk-windows-x64.zip`** from the link above.
2. Right-click the downloaded `.zip` file and select **Extract All...**
3. Open the extracted folder and double-click **`hydrawalk_flutter.exe`** to run the app instantly!
